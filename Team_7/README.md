Exam-cheating-detection-using-AI-cameras
This project is an AI-driven exam cheating detection system using computer vision. It leverages YOLOv8 and head pose estimation to identify abnormal head movements and phone usage. Normal forward-facing posture is marked Non-Cheating, while looking around or using a phone is flagged as Cheating. Project Overview
